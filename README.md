# Roll Drop

### Unity Mobile Hyper Casual Game Project

Mobile hyper casual game based on fast and fun mechanics, designed with Unity and URP.

## Key Features

- AdMob Integration
- Sound Effects Included
- Touch Screen Support
- Easy Level Design Workflow
